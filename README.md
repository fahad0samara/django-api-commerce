# E-Market

A full-featured e-commerce platform built with Django.

## Features

- User authentication and authorization
- Product catalog with categories
- Shopping cart functionality
- Order management
- Payment integration (Stripe)
- Wishlist system
- Review system
- Real-time notifications
- Admin dashboard
